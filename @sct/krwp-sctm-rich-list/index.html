KRWP 서비스 추가
  • 금일부터 KRWP 보유량에 따라 @sct.krwp 에서 보팅오던 서비스가 중단되고, KRWP를 전송하여 전송된 만큼 스팀과 SCT 보팅이 오는 서비스로 변경되었습니다.상세 내용 링크
  • 이와 동시에 스팀코인판 사이트에서 쉽게 서비스를 이용할 수 있게 하기 위해서 UI의 변경이 있었습니다.
  • 포스팅 상세페이지 우측 하단에, 기존에 “홍보”버튼이 있던곳 좌측에 “KRWP 서비스” 버튼이 추가 되었습니다.

  • 클릭을 하면 하단의 이미지가 나타나고, KRWP서비스 요청에 사용할 KRWP 수량을 입력 후 “KRWP 서비스” 버튼을 클릭하시면 됩니다.

  • 전송 여부는 우측 링크에서 확인하시길 바랍니다. 전송여부 확인

  • 보팅을 위해 전송된 KRWP 는 다음과 같이 사용될 예정입니다.

50% : SCTM 홀더들에게 분배
40% : 소각
5% : 이벤트
5% : 개발기금
SCTM 스테이킹한 사용자에게만 KRWP가 분배되며, 주 1 회 분배됩니다.

SCTM 환매서비스 버튼 추가
  • 스팀코인판 메인화면 우측에 SCTM 환매서비스 버튼이 추가 되었습니다.SCTM 환매 관련 상세내용 링크
  • 클릭시 하단 이미지와 같이 SCTM 환매서비스 요청창이 뜨고 참여할 수량을 입력 후 “SCTM 환매 요청”을 클릭하시면 요청됩니다.
Rich List 링크 변경
  • 기존에 Steem-Engine 에서 제공해주던 Richlist가 제대로 작동하지 않아, @jacobyu 님이 개발한 사이트로 이동됩니다.
  • Holder List를 클릭시 현재 보유리스트를 확인 하실 수 있습니다.
추가로 궁금한 사항이나 문의 사항 있으시면 댓글 남겨주십시오.
