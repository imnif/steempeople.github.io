  • sctswap 유동성풀 중 하나인 @sct.kpsteem 계정에 하이브가 스냅샷되었습니다.

  • 사전에 스냅샷 될거라는 예상을 못했기에 처분 방침을 사전에 공지해드리지 못했습니다.

  • 전일 하이브를 전량 스팀으로 전환하였습니다.

  • 전환된 스팀수량은 21,242.279스팀입니다.

  • 이것을 sctswap에서 전량 krwp로 스왑하였습니다. krwp 가치도 안정시키고 유동성 공급자의 지분도 그대로 유지시키기 위한 조치입니다.

  • 6,558.309 krwp로 스왑되었습니다.

  • 스왑된 krwp는 유동성 공급자 지분에 비례하여 유동성 공급자에게 전송될 예정입니다.

[공지] 암호화폐 교환 기능 추가

스팀코인판 우측에 암호화폐 교환 기능이 추가되었습니다. 기존에 블럭트레이드가 운영하던 사이트와 거의 동일하다고 보시면 됩니다.

교환 기능 화면

  • PC버전인 경우 우측 사이드
  • 모바일의 경우 우측 상단의 햄버그 메뉴를 클릭해보시면 제일 하단에 “Exchange Now” 메뉴를 이용하시면 됩니다.

상세 사용법(아래 링크에 자세한 사용법이 있습니다.)

  • @sct.krwp님 의 글
  • @happyberrysboy님 의 글

이런 분들께 추천드립니다.

  • 코인 종류를 바꿔야 하는데 거래소를 이용하기가 싫은 경우.(거래소로 보내고, 팔고, 다시 사고, 다시 전송 단계를 없애줌)
  • 내가 보유하고 있지 않은 코인을 다른 사람에게 다른 코인을 이용하여 주려고 하는 경우
    : 예시) 상대방은 이더리움을 받기를 원하지만, 나는 비트코인만 있는경우 비트를 전송하고 받을 이더 주소를 상대방 주소로 입력

[공지] @sct.krwp 계정 보팅 누락에 대해 사과드립니다.

  • 원인을 알 수 없는 이유로 @sct.krwp 계정 보팅 누락이 생기는 경우가 발생한다는 제보를 받았습니다.

  • 보팅누락이 발생하지 않도록 개발팀이 최선을 다하고 있으나, 여러가지 이유로 완전한 해결이 어려운 상황입니다.

  • 스팀블록체인과 스팀엔진에 종속되는 니트로스 입장에서 현실적으로 이것을 완전히 해결할 수 있다는 말씀도 드리기 어렵습니다.

  • 향후에도 이러한 상황은 지속적으로 발생할 가능성도 있으며, 발생하지 않을수도 있습니다.

  • 현재 스팀코인판은 이 문제를 완전히 해결할 만한 능력을 가지고 있지 못합니다.

  • 이 부분을 완전히 해결할 능력이 없음을 말씀드리며, 더불어 이 문제를 경험하신 분에게는 죄송하다는 말씀을 전합니다.

죄송합니다.

[SCTSWAP 업데이트] token swap 확장

token swap 확장 업데이트

기존에는 스왑시스템에 리스팅된 토큰은 오직 KRWP 토큰과 교환할 수 있었습니다.

이번 업데이트를 통해, 리스팅된 토큰들끼리 교환할 수 있습니다.

ex)

  • SCT <-> STEEM
  • SVC <-> SCTM

SCT에서 STEEM으로 교환 할 때는 아래와 같은 과정으로 교환됩니다.

  1. 사용자가 SCT -> STEEM 교환 요청
  2. 스왑시스템 내부적으로 SCT를 KRWP로 교환
  3. 2번에서 교환된 KRWP를 STEEM으로 교환
  4. 사용자에게 전송

사용 방법

  • 스팀코인판 스왑은 사이트의 오른쪽 사이드바에서 SCT SWAP 버튼
  • https://www.steemcoinpan.com/market 에 접속
