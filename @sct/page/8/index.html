[공지] 하급 뽑기판 오픈

하급 뽑기판이 오픈되었습니다.

링크 : https://apisct.cloud/manager
링크로 가셔서 뽑기 - 하급뽑기판을 누르면 나타납니다.

1 ) 뽑기권 구매
  • 무료 뽑기판을 제외한 뽑기판은 뽑기권을 구매하셔야 참여하실 수 있습니다.
  • 뽑기권은 가장 아래와 같이 우측 메뉴에서 구매 가능합니다.
  • 뽑기권을 구매하기 위해서는 먼저 로그인이 이루어져야합니다.(키체인을 이용)
  • 뽑기권 종류를 선택하고, 수량을 입력 후 “구매”버튼을 클릭하면 구매가 이루어집니다.
  • 뽑기권으로 모아진 SCT의 50%는 소각, 50%는 다음 로또 상금에 추가가 됩니다.(추후 변경될 수 있습니다.)
2 ) 뽑기참여
  • 로그인을 하셔야 뽑기에 참여가 가능합니다.
  • 아래와 같이 현재 뽑기판에 남아 있는 잔여보상이 표시되고, 원하시는 번호를 클릭하면 해당 번호가 뽑아집니다.
  • 결과는 바로 팝업창으로 표시가 되게 됩니다.

3) 뽑기보상
  • 아직은 뽑기권 사용페이지는 없습니다. 최대한 빠른시일내로 공정하게 뽑기를 할 수 있도록 하겠습니다.
  • 현재 하급 뽑기판에는 총 3종류의 보상이 있습니다.
  • 뱃지장착 : 뱃지를 장착 할 수 있는 보상입니다. 현재 바로 사용 할 수는 없으나, 조만간 사용방법에 대해서 공지 할 예정입니다.
  • 보팅 : 특정 계정으로부터 보팅을 받을 수 있는 보상입니다. 사용하시려면 뽑기결과/보상사용/뽑기권구매 메뉴로 가서 사용하시면 됩니다.
  • 로또참여권 : 로또에 참여 할 수 있는 보상입니다. 로또는 주사위를 굴려서 현재 모인 상금을 일정 비율만큼 타갈 수 있는 참여권입니다.
주사위로 나온 숫자 상금에서 타가는 비율
100 80%
99 30%
98 20%
97 10%
96 1%

예) 현재 쌓여 있는 상금이 모두 1000 SCT인 경우

  • 주사위 100이 나오면 상금 800SCT
  • 주사위 99가 나오면 상금 300SCT
  • 주사위 98이 나오면 상금 200SCT
  • 주사위 97이 나오면 상금 100SCT
  • 주사위 96이 나오면 상금 10SCT

최초 상금은 1000 SCT에서 시작하고, 여기에 뽑기권 판매금액만큼 누적됩니다. (뽑기권 50%는 소각)

궁금하신 사항이나 문의사항 있으면 답글 주십시오. 감사합니다.

[공지] KRWP 베네피셔리 서비스 재가동

1월 13일 중단되었던 KRWP 베네피셔리 서비스가 금일 UTC 6시부터 재가동 되었습니다.
그동안의 작업 내용은 다음과 같습니다.

1 ) SCT 관련 블록 정보 저장 봇 로직 변경

  • SCT 관련 custom json / @sct.krwp 베네피셔리 설정 포스팅 정보와 스팀 보상 정보를 분리하여 저장
  • custom json 정보 중 불필요한 정보 제외(claim, auto cliam 등)
  • 마지막 블럭과의 차이에 따라 한번에 처리하는 블럭 개수를 가변적으로 처리하도록 수정
    추후 장애가 발생할 경우 빠르게 복구가 가능함
    예) 1000 건 이상 차이날 경우 10ms 마다 100건 씩 입력, 30건 이하 차이날 경우 3000ms 마다 1건 씩 입력

2 ) KRWP 베네피셔리 봇 로직 변경

  • 분리된 저장소로 정보를 읽어서 처리하도록 수정

현재 SCT 보상은 모두 처리가 되었고 스팀 보상은 2일 정도의 데이터가 처리되지 않았습니다.
내일 중이면 모든 보상에 대해서 전송 처리가 완료될 것입니다.

[공지] 스팀 엔진 크래시 이슈 및 sct.krwp 베니피셔리 시스템 점검

  • 지난 주말에 발생한 스팀 엔진 노드 크래시 발생으로 인해 스콧봇과 스판 니트로스 이용이 제한되었습니다

  • 이번 이슈 발생 기간 동안 페이아웃된 글 중 krwp 저자 보상, @sct.krwp 베네 설정 등에서 문제가 발생한 글들이 발견되었습니다. (예시: 0.001 SCT 미만이 아닌데 저자 보상으로 SCT 가 지급된 경우)

  • 이와 관련해, 스팀 코인판에서는 krwp 저자 보상 로직, @sct.krwp 계정 베네피셔리 설정 관련된 부분에 대해 전체 점검 및 업데이트 작업을 병행하고 있습니다.

  • 관련 작업이 완료되기 전까지 @sct.krwp 계정에 100% 베니피셔리를 설정한 글에 대한 보상이 지연될 수 있으며, 작업이 완료되는대로 지연된 보상은 순차적으로 지급될 것입니다.

  • 작업 완료 시점은 별도 공지 예정입니다. 이용에 불편을 드려서 죄송합니다.

[공지] 스팀코인판 광고 시스템 도입 예정

  • sct 토큰에 광고토큰으로서의 유틸리티 기능을 추가하려고 합니다.

  • 스팀코인판 니트로스 최상단에 위 이미지와 같은 레퍼럴 URL이 연결되는 텍스트 광고 시스템을 운영하려고 합니다.

  • 월간 광고료는 2,000sct로 잠정 결정하였으며, 광고 수요가 많을 시 광고료는 인상될 수 있고 랜덤 노출방식으로 운영이 변경될 수 있습니다.

  • 광고료는 전액 소각하여 커뮤니티 구성원 모두에게 광고 이익이 돌아가도록 합니다.

  • 모바일 버전의 경우 위와 같이 표시됩니다.

광고주가 되는 방법

  • 댓글로 광고 문의를 해 주십니다.
