안녕하세요, 디온(@donekim)입니다. **EOSIO 코드 포크체인 텔로스(TELOS)에서 생태계의 확장을 위해 무료 계정생성 서비스를 지원하기 시작**했습니다. 그동안 이오스 메인넷의 제네시스 계정이 없어서 텔로스 메인넷에 접근하실 수 없었던 유저분들은 **TLOS토큰을 소유하고 있지 않더라도 텔로스 계정을 무료로 생성**하실 수 있습니다. 텔로스 계정의 무료 생성방법은 다음과 같습니다.

![telos logo1.jpeg](https://cdn.steemitimages.com/DQmZJM5KAVJJhPEQwJzdu8irKsFSBii9769x6BaetpLfEmK/telos%20logo1.jpeg)

텔로스 계정을 무료로 생성하기 위해서는 먼저 **텔로스의 공식 지갑 중 하나인 스쿼럴 월렛(Sqrl wallet)의 0.5.5 버전을 다운로드** 받아야 합니다. `(0.5.5 이전 버전을 사용하고 계신 분들은 반드시 업데이트를 해주셔야 합니다)` 다운로드 또는 업데이트를 위해 텔로스의 공식 다운로드 홈페이지(https://telosfoundation.io/downloads)에 접속해주세요.

> 기존에 스쿼럴 월렛에 계정 정보를 입력하여 사용 중이신 분들의 경우, **0.5.5 버전으로 업데이트를 하면 기존에 저장되어 있던 계정정보가 초기화**되므로 필요한 경우 반드시 계정 정보를 백업해놓으시기 바랍니다.

![download.png](https://cdn.steemitimages.com/DQmUaua21UttDuKk9FoEzNNvoqSieASedC8taJKbt4AFPUk/download.png)

위와 같은 화면이 나타나면 **자신의 운영체제에 맞는 버튼을 클릭**해주세요.

![sqrl.png](https://cdn.steemitimages.com/DQmaoun8bzvUi16FzwGGQM7PDLUuXCiNPnu3E5NnvtEXJT1/sqrl.png)

해당 버튼을 클릭하면 위와 같이 [Sqrl wallet의 깃헙(github)사이트](https://github.com/Telos-Foundation/Sqrl/releases)로 넘어가는 것을 확인하실 수 있습니다. 여기서 마우스 스크롤을 내려서 **다운로드 목록을 확인**해주세요.

![download list.png](https://cdn.steemitimages.com/DQmUqjzUyFhvzLTEoFDvjHu3praDpe9DLdi14XKPRabzYUC/download%20list.png)

**윈도우 사용자**들의 경우에는 `win-Sqrl-0.5.5.exe`파일을 클릭하여 다운로드 받아주시고, **맥 사용자**들의 경우에는 `mac-Sqrl-0.5.5.dmg` 파일을 클릭해서 다운로드 받아주세요. 파일 다운로드가 모두 완료되고 나면 해당 파일을 실행하여 스쿼럴 월렛을 설치해주세요.

![create1_check.png](https://cdn.steemitimages.com/DQmPzN75SBUF97vGVAAfFdpVD5rR4xK7DSjY8vhowbeRB5S/create1_check.png)

스쿼럴 월렛의 설치가 완료된 후 해당 파일을 실행하면 다음과 같은 화면이 나타나는 것을 확인하실 수 있습니다. 우리는 **신규 계정을 생성할 예정이므로 `Create New Account`라는 버튼을 클릭**해주세요.

![create2.png](https://cdn.steemitimages.com/DQmf6DZwwvqKf4z4D1mbeQBotBKk2DznCJysux6ezoSBVuC/create2.png)

그러면 위와 같이 새로운 계정의 정보를 입력하는 화면이 나타나는 것을 확인하실 수 있습니다. 이오스 메인넷 계정과 마찬가지로 계정을 생성하기 위해서는 **①계정이름(Account name), ②오너 권한의 퍼블릭키(Owner Public key), ③액티브 권한의 퍼블릭키(Active Public key)**를 넣어주어야 합니다.

> 다들 잘 알고 계시겠지만, **계정이름(Account name)은 반드시 A부터 Z까지의 알파벳과 숫자 1부터 5까지의 조합**으로 만드실 수 있습니다. 꼭 알파벳과 숫자를 모두 사용해야 할 필요는 없으며 **알파벳 only** 또는 **숫자 only**로도 계정이름을 설정하실 수 있습니다. 더불어 이미 다른 사용자가 사용 중인 계정과 동일한 이름은 사용하실 수 없으므로 **계정을 생성하기 전에 텔로스 익스플로러(https://telos.eosx.io/)에서 생성하고자 하는 계정이름을 다른 사용자가 사용하고 있는지를 확인**하시기 바랍니다. 

혹시라도 키쌍(key pairs)과 액티브키(Active key), 오너키(Owner key)의 개념에 대해서 아직까지도 헷갈리시는 분들은 아래의 포스팅을 참고해주세요.



- [이오스 키쌍(Key pairs) 및 권한(Permission)에 관한 설명](https://steemit.com/dclick/@donekim/-public-key-private-key-owner-key-active-key--1540137295651)
- [액티브키(Active key) / 오너키(Owner key)에 관한 설명](https://steemit.com/dclick/@donekim/-active-key-owner-key-1-1541752457800)

![create3_check.png](https://cdn.steemitimages.com/DQmYJYuekEZxo9QsA2yQhufv5G5yKMmn35UHewf4cqBn2b7/create3_check.png)

**계정이름(Account name)은 위와 같이 12자리로 직접 입력**하시면 되고, **오너키와 액티브키는 위의 화면에서 붉은색 박스로 표시되어 있는 새로고침 모양의 화살표를 클릭**하시면 랜덤으로 키쌍이 생성됩니다. 모두 입력이 완료되었으면 `Next`버튼을 클릭해주세요.

![create4_check.png](https://cdn.steemitimages.com/DQmVAUyUd4m7XQPcSfAVTAnPpkcaugv1i92wEYb1At5Mmxf/create4_check.png)

그 다음은 **방금 위에서 생성한 오너키와 액티브키의 키쌍(퍼블릭키, 프라이빗키)을 저장하는 단계**입니다. `Copy keys / Already Have Keys`라는 버튼을 클릭해주세요. 해당 버튼을 클릭하면 키쌍(퍼블릭키, 프라이빗키)의 정보가 자동으로 복사가 됩니다. 그 다음에는 **메모장 등 본인만이 접근가능한 곳에 붙여넣기(ctrl+v)를 하여 안전한 곳에 키쌍 정보를 잘 저장**해주세요. `(방금 생성한 오너키와 액티브키에 연결된 프라이빗키 정보가 들어있기 때문에 안전한 곳에 잘 저장해두셔야 합니다)`

![create5_check.png](https://cdn.steemitimages.com/DQmYWsTn2jb5n3aWqiohCmnobySAx1PmMgApFW8SAJR7fqA/create5_check.png)

`Copy keys / Already Have Keys`라는 버튼을 클릭해서 키쌍 정보를 안전한 곳에 잘 저장하고 나면 화면 하단에 `Keys Are in Safe Place`라는 버튼이 활성화됩니다. 해당 버튼을 클릭해주세요.

![create6_check.png](https://cdn.steemitimages.com/DQmX28DTiVhYgQtVo6C4TUxR1HokSb3cj2c2KeTZTsxYarv/create6_check.png)

**계정 생성을 신청하기 전에 최종적으로 입력한 정보들이 맞는지를 확인하는 단계**입니다. 계정이름이 잘 입력되었는지, 그리고 오너키 또는 액티브키가 저장해놓은 것과 동일한지를 마지막으로 체크하신 뒤에 이상이 없는 경우 `Create My Account`라는 버튼을 클릭해주세요.

![create7_check.png](https://cdn.steemitimages.com/DQmRNLanmGDsRxx4PwyvrKufLVj3CdWzDeayZe17CaTZZ49/create7_check.png)

**위와 같은 메시지(Your account has been created!)가 나타나면 계정생성이 정상적으로 완료된 것**입니다. 혹시라도 정상적으로 생성되었는지 다시 한 번 확인하고 싶으신 분들은 텔로스 블록 익스플로러(https://telos.eosx.io/)에 방금 생성한 계정이름을 입력해보시면 됩니다.


![create8_check.png](https://cdn.steemitimages.com/DQmPRadmXRYasrzPqvkgRxMGVDL1G9MrvNfNQfT5M19C4D9/create8_check.png)

이와 같은 방법을 통해 여러 개의 계정을 생성하려고 시도하는 분들이 있으실 수 있는데, **스쿼럴 월렛을 통해 무료 계정생성은 딱 1번만 지원**됩니다. 추가로 무료 계정생성을 시도하는 경우에는 위와 같은 메시지가 나타나며 계정생성이 되지 않습니다 :)

본 텔로스 무료 계정생성은 제네시스 계정이 없어서 텔로스 메인넷에 접근하지 못하는 분들을 위해 텔로스에서 진행하는 특별 행사로, **선착순으로 최대 1,000,000개까지만 지원할 예정**이라고 하니 포스팅 내용을 참고하시어 텔로스 계정이 없으신 분들은 이참에 계정을 꼭 생성해놓으시기 바랍니다. 감사합니다 :D
