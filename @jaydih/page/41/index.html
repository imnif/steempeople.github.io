이오스나이츠 : 극한의 확률








레어 나올 확률 : 10.951% 노멀 제외 환산 : 88.379%
레어 펫의 종류 : 8가지 아기용이 나올 확률 : 11.047%
아기용이 연속 5번 나올 확률 : 11.047%^5 = 0.00165% 고대 펫이 나올 확률 : 0.003%
* 노멀 제외 환산 : 0.024%

나는 고대 펫 보다 15배 힘든 확률로 아기용 다섯 마리를 획득했다.
