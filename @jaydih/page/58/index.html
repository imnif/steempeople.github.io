스팀월드를 통한 스팀 무료 계정 생성 (RC 소모)

스팀파워를 충분히 보유한 사람이라면 스팀월드를 통해 매우 쉽게 RC를 사용하여 무료 계정을 생성할 수 있다. 사용 방법도 너무 쉬워 어렵게 포스팅 해볼 필요도 없어보인다.

아래 주소를 주소창에 붙여넣고 자신의 아이디를 넣어 접속해보면 싸이트 하단에 Account Creator를 찾아볼 수 있다.

  • 스팀월드
  • https://steemworld.org/@본인아이디

  1. Claim Account 버튼을 통해 계정 생성 토큰을 수령
  2. Account Name에 신규 계정명 입력
  3. Generate 버튼을 통해 마스터키 발급
    (이 때 반드시 키를 잘 저장해두자,
    반드시!
    )
  4. Create Account 버튼을 눌러 신규 계정 생성

오늘자 기준으로 계정 생성에 필요한 RC는 7400G RC정도로 대략 3800 스팀파워 보유자라면 계정을 쉽게 생성할 수 있다. 다만, 각 활동에 필요한 RC비용은 실시간 유동적으로 지난 포스팅을 통해 자세한 비용을 확인 할 수 있다.

  • RC 비용과 가격 변화 그래프

스팀잇 유용한 정보

  • 크롬 확장 프로그램 “SteemPlus”
  • RC로 무료 계정 생성 할 수 있는 토큰 만들기
  • RC로 신규 계정 만들기
  • RC 비용과 가격 변화 그래프
  • 누가 내 글에 가장 많이 기여했을까?
  • 스팀파워 손쉽게 임대하기
  • 포스팅 권한 삭제
  • 커뮤니티 기반 SMT 물 밑 작업 HF20
  • SMT 다음은 “스팀 마켓 플레이스!”
  • 6월자 스팀 백서 정독 후기
  • 스팀잇 빨간 글씨
