RC로 신규 계정 만들기 (HF20)

HF20 도입 이후 RC(Resource Credit)을 통해 무료(?)로 계정 생성이 가능해졌다.
우선, RC를 통해 계정생성을 위해서는 ‘Discounted Account Claim’이라는 것을 진행하여 무료 계정 생성을 위한 토큰을 준비해 두어야한다.

  • 요기를 클릭하면 스팀컨넥트를 통해 비교적 안전한 방법으로 토큰 생성을 진행 할 수 있다.

앞서서 이전 글에 스팀컨넥트를 사용하여 ‘Discounted Account Claim’을 알아보았는데, 아직 스팀컨넥트에서 해당 토큰을 이용해 최종 계정 생성까지는 지원하지 않는듯하여 다른 방법을 찾아보았다.

그 결과… 개발자가 아닌 일반인으로써 사용할 수 있는 방법은 현재로써 두가지 정도인듯하다.
두가지 방법 모두 자신의 아이디와 액티브키를 직접 입력해야하므로 사용시 각별히 주의 바란다.

  • 첫번째 방법 - Steem HF20: Account Create
    @reazuliqbal 에 의해 만들어진 것으로 github의 웹페이지 형식으로 올라와 있어 비교적 쉽게 계정 생성 작업이 가능하다. (github이란 일종의 개발자 포럼으로 이 곳에 검증을 위한 오픈소스가 전부 올라온다)
    해당 툴의 원문 설명 포스팅은 이 곳을 클릭하여 참조하길 바라며 스팀의 증인 중 한분인 @utopian-io 에 의해 검증된 것으로 보인다.

    Account Username : 새로 만들고자 하는 계정 아이디
    Password : 마스터 키 / 페이지 새로고침을 통해 마스터 키를 재발행 할 수 있다
    (반드시 이 키를 저장해두어야한다)
    Creator Account : 신규 계정을 생성하는 계정
    Delegation : 원한다면 스파 임대까지 함께 해줄 수 있으나 0으로 둬도 된다



  • 두번째 방법 - create_and_claim_hf20.html
    @mcfarhat 에 의해 만들어진 것으로 html 스크립트를 모두 오픈소스로 github과 자신의 포스팅을 통해 공개해 두었다.
    텍스트 문서에 해당 스크립트를 모두 복사하여 붙여넣기하고 html 확장자로 저장 후 자신의 브라우저에서 열면 된다. 그게 어렵다면 github에서 직접 다운로드 할 수 있으며 해당 툴의 원문 설명 포스팅 또한 링크를 통해 확인하길 바란다.

github을 통해 소스코드를 다운로드하여 zip으로 압축된 파일을 풀면 html 파일을 찾을 수 있고 그 파일을 자신의 브라우저에 드래그&드롭하면 아래와 같은 화면이 나온다. 비교적 심플하고 간단하며 체크박스를 통해 Claim(토큰생성) 과 Create(계정생성)를 진행할 수 있다.

New Account Pass : 신규 계정에 할당될 마스터키를 암호화하기 위한 임시 비밀번호이지만 꼭 기억해두길 바란다
