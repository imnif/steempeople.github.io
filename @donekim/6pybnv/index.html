[이오스 메인넷 런칭] 이오스 참고사이트 한 눈에 모아보기

안녕하세요, 디온(@donekim)입니다. 이오스 메인넷 런칭에 따라 관련된 참고사이트들을 한 눈에 모아서 보실 수 있도록 정리해보았습니다. 이오스 계정이름, 계정별 보유 수량, 트랜잭션, 블록생성정보, 블록프로듀서 후보 정보 등에 대해서 정보를 제공하는 다양한 사이트들이오니 즐겨찾기 해 두시고 참고해주세요!


[이오스 참고사이트 한 눈에 모아보기]



- https://eospia.com/ : EOS계정 조회사이트 - 이더리움 지갑 주소로 EOS 계정이름, 퍼블릭키, 보유수량, 보유량 순위 등 확인 가능




- https://bp.eosgo.io/ : EOS 블록프로듀서 조회 사이트 - 각 블록프로듀서 후보들의 상세 정보 조회 가능 (by EOSGO)





- https://eosflare.io/ : EOS 트랜잭션 조회 사이트 - 계정이름 또는 퍼블릭키 등을 통해 계정정보, 트랜잭션 조회 가능 (by EOSflare)



- https://portal.eoseoul.io/ : EOS 투표포털 및 BP리스트 확인 사이트 - 스캐터(Scatter)를 이용하여 투표에 참여하거나 상위21개 BP를 포함한 전체 BP리스트 확인 가능(by EOSeoul)







- https://eospark.com/ : EOS 트랜잭션 조회 사이트 - 블록프로듀서 리스트 확인, 트랜잭션 및 계정 정보, 이오스 가격변화 등 조회 가능




- https://eostracker.io/ : EOS 트랜잭션 조회 사이트 - 블록, 트랜잭션, 거래, 계정, 블록프로듀서 등 확인 가능








- https://eosauthority.com/voting : EOS 전체 투표율 조회 사이트 - 현재 투표율 한 눈에 확인 가능 (by EOS Authority)




- https://www.eosstats.io/hourlystats/ : BP리스트 확인 사이트 - 시간별로 BP순위 변동을 확인할 수 있으며, 특정 BP 클릭 시 해당 BP에 클릭한 계정 확인 가능 (by EOS New York)






- https://toolkit.genereos.io/tools : EOS 스테이킹 관리 및 계정생성 사이트 - 스캐터(Scatter)를 이용하여 스테이킹을 관리하거나 12자리의 영문+숫자 조합 자녀계정 생성 가능 (by GenerEOS)





- https://www.myeoskit.com/ : EOS 투표포털 및 계정생성 사이트 - 스캐터(Scatter)를 이용하여 투표에 참여하거나 12자리의 영문+숫자 조합 자녀계정 생성 가능 (by EOS Asia)
