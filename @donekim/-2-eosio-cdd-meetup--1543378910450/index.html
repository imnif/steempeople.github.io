[이오스 이야기] 제2회 EOSIO CDD MEETUP에 많은 관심 부탁드립니다

안녕하세요, 디온(@donekim)입니다. ‘3 Sec. Smart Communication’ TOOKTOOK에서 주최하는 제2회 EOSIO CDD MEETUP이 이틀 앞으로 다가왔습니다. 현재까지 317분께서 참가신청을 해주셨고 최대 400명까지 참여가 가능한 관계로 아직까지 좌석에 여유가 조금 있습니다. 혹시라도 관심 있으신 분들께서는 꼭 참여하셔서 직접 얼굴도 뵙고 인사드릴 수 있었으면 좋겠습니다 :) 참여를 희망하시는 분은 여기를 클릭해서 참가신청을 해주시면 됩니다.

[주제]
Happy New EOS 2019

[개요]

  • 일시: 2018년 11월 30일 금요일, 저녁 7시
  • 장소: 삼성동 코엑스 메가박스 3관
  • 정원: 400명 *무료
  • 주최: TOOKTOOK
  • 주관: PUBLYTO
  • 후원: DABBLE, DEXEOS

[발표자]
• Session 1 Community

  • Devin Lee, EOS & KOREOS Evangelist
  • Done Kim, EOS Journalist
  • Geonpil Jeon, EOS Supporter
  • MAX Cho, EOS Activist

• Session 2 dApp/DAC/Infra Firms

  • Sungho Seo, Technical Manager, TOOKTOOK
  • Alex Lee, CSO, DEXEOS
  • Hoon Lim, UXUI Designer, PUBLYTO
  • Daniel Zhang, CMO, Chaince

• ‘Allied Session’

  • David Margulies, EOS Alliance & Emergency Committee

• 네트워킹

[시간표]
18:40 - 19:00 리셉션(신청확인, Gift, 음식과 음료)
19:00 - 19:05 환영사
19:05 - 20:05 연사발표: Session 1
20:05 - 20:20 휴식, 네트워킹
20:20 - 21:10 연사발표: Session 2
21:10 - 21:20 연사발표: ‘Allied Session’
21:20 - 22:00 AMA
22:00 - 22:30 네트워킹

[참가혜택]
선착순 참가자 300명에게 3만원 상당의 향초 + ⍺
*밋업 사전 신청자에 한함

[상세문의]
TOOKTOOK Telegram Group : https://t.me/tooktook_KR

[유의사항]
• 400명 정원이며 조기 마감될 수 있습니다.
• +게스트를 입력하신 경우 반드시 모든 게스트 분들을 동반하셔야 입장이 가능합니다.
• 코엑스몰 주차는 4시간 기준 4,800원입니다. 입장 시 제공받은 티켓 지참 후 극장 매표소에서 주차확인을 받으시면 됩니다.
• 연사 명단과 발표주제는 소폭 변경될 수 있습니다.
• 저녁식사 대용으로 간단한 음식과 음료가 제공됩니다.
• 본 밋업은 시세예측, 차익거래, 트레이딩 등 매매정보를 다루지 않습니다.
• 신청 후 부득이 불참하시는 경우에는 반드시 취소 버튼을 눌러주세요. 노쇼 명단은 엄격히 관리되며 추후 본 밋업과 관련된 모든 행사에 참여하실 수 없습니다.

[디온의 이오스 이야기 시리즈 한 눈에 보기]

#1. 이오스 생태계 초심자를 위한 기초설명

(1) 기존 암호화폐(비트코인/이더리움)와 이오스의 지갑생성 차이점 설명

  • 이오스 계정 및 자원에 대한 설명

(2) 이오스 계정 생성 방법 설명

  • 이오스 생태계에서 멀티계정이 필요한 이유
  • 1편 : 기존 계정 보유자
  • 2편 : 기존 계정 미보유자
  • 3편 : 이더리움 지갑 ERC-20토큰 당시 미등록홀더

(3) 스캐터 데스크탑 설치 및 계정등록 방법 설명

(4) 이오스 지갑들의 종류 및 특징

(5) 이오스 키쌍(Key pairs) 및 권한(Permission)에 관한 설명

(6) 액티브키(Active key) / 오너키(Owner key)에 관한 설명

(7) [토큰 배분 메커니즘에 관한 설명]

  • 1편 : 새롭게 등장한 에어그랩(Airgrab)이란 무엇인가?
  • 2편 : 에어드랍(Airdrop), 쉐어드랍(Sharedrop), 에어그랩(Airgrab)
  • 3편 : 에어드랍(Airdrop), 에어그랩(Airgrab), 스카이훅(Skyhook)

#2. 툴킷 설명 및 종류별 사용방법

(1) 툴킷(Toolkit)에 대한 설명

(2) 이오스 허브(eoshub.io) - by EOSYS 사용방법

(3) 마이이오스킷(myeoskit.com) - by EOS ASIA 사용방법

(4) 블록스닷아이오(bloks.io) - by EOS Cafe Block & HKEOS 사용방법

  • 1편 : 메뉴의 구성 소개
  • 2편 : 토큰전송 및 에어그랩
  • 3편 : 신규 계정(자계정) 생성방법
  • 4편 : 오너키/액티브키 변경방법

#3. 이오스 댑(DApp) 생태계 자세히 살펴보기

(0) 이오스의 댑 생태계 훑어보기 시리즈

  • 1편 : 전자상거래 / 교육 / 거래소 / 핀테크
  • 2편 : 게임 / 의료서비스 / 신원인증
  • 3편 : 마켓플레이스 / 미디어
  • 4편 : 플랫폼 / 보안 / 소셜 / 교통

(1) 아티디움(Atidium) : 이오스 기반 자산관리 매니저

(2) 호루스페이(HORUSPAY) : 이오스 기반 글로벌 페이롤 포탈

(3) 덱시오스(DEXEOS) : 이오스 기반 탈중앙화거래소

(4) 친타이(Chintai) : 이오스 자원임대 플랫폼

  • 1편 : 친타이 소개
  • 2편 : 친타이 사용방법

(5) 데블(Dabble) : 이오스 기반 소셜 미디어 플랫폼

(6) 파슬(Parsl) : 합법적인 대마초 공급망 관리 플랫폼

(7) 하이어바이브즈(HireVibes) : 크라우드 소싱 기반 탈중앙화 채용 플랫폼

(8) 센스챗(SenseChat) : 이오스 기반 탈중앙화된 P2P 메신저

  • 1편 : 센스챗에 대한 설명
  • 2편 : 센스챗 베타테스트 시작 안내 및 후기

(9) 이오스나이츠(EOSKnights) : 이오스 기반 방치형RPG 게임

(10) 리들(RIDL) : 이오스 기반 탈중앙화된 평판 및 신원관리 시스템

  • 1편 : 리들 백서 읽어보기
  • 2편 : 리들 백서 읽어보기
  • 3편 : 리들 백서 읽어보기

(11) 카르마 : 이오스 기반 탈중앙화된 선행보상 소셜 플랫폼

  • 1편 : Do Good, Earn KARMA 선행보상 소셜 플랫폼 카르마 알아보기
  • 2편 : 카르마 퍼블릭베타(KARMA Public Beta) 테스트 참여방법 및 후기

(12) 루미오스(Lumeos) : 탈중앙화된 소셜 서베이 플랫폼

(13) 엠앤에잇(Emanate) : 자치적으로 운영되는 탈중앙화된 음원 거래 플랫폼

  • 1편 : 엠앤에잇 디앱에 대한 설명

  • 2편 : 엠앤에잇의 EMT토큰 및 토큰 모델에 대한 설명

#4. EOS.IO 코드 포크체인 살펴보기

(1) 포크체인(Forkchain)에 대한 설명

(2) 메인체인과 포크체인의 비교 설명

(3) 텔로스(TELOS)

  • 1편 : 텔로스 백서 살펴보기
  • 2편 : 텔로스 백서 살펴보기

(4) 워블리(WORBLI) : 기업 금융 시스템 운영에 최적화된 시스터체인

  • 1편 : 새로운 포크체인 워블리와 셰어드랍에 대한 설명

  • 2편 : 워블리 포털 계정 생성 방법에 대한 설명
