[EOS Inside] 스캐터 데스크탑 신규 버전 Meteoric 출시!



Image Source : Scatter Blog


안녕하세요, 디온(@donekim)입니다. 많은 유저들이 오랫동안 기다려왔던 스캐터 데스크탑의 신규 버전 Meteoric이 어제 출시되었습니다. Meteoric이라는 수식어에 걸맞게 새롭게 출시된 스캐터는 비약적인 발전을 보여주며 한 단계 더 진화된 섹시한 기능들을 탑재하고 돌아온 것 같습니다.

- Scatter Meteoric 다운로드 : https://get-scatter.com/download





Image Source : Scatter Blog


무엇보다 눈에 띄는 점은 비약적인 UI의 개선입니다. 사실 지금까지의 스캐터 데스크탑은 초보자들이 사용하기 어렵게 느껴지는 유저 인터페이스가 가장 큰 문제점 중 하나로 손꼽혔으나, 이번에 출시된 Scatter Meteoric에서는 초보자들도 직관적으로 이용할 수 있도록 대규모 UI 업그레이드가 진행되었습니다.




개인적으로 지금까지의 스캐터 데스크탑의 디자인 중 이번 버전이 가장 마음에 드네요.




이번 신규 버전의 가장 대표적인 업데이트 중 하나는 EOSIO, 이더리움, 트론 뿐만 아니라 비트코인 지갑 기능까지 통합이 되었다는 것입니다. 이제는 별도의 BTC월렛을 사용할 필요 없이 스캐터를 통해 간단하게 주소를 생성하고 BTC자산을 관리할 수 있습니다.




평판 및 신원정보 레이어 RIDL도 스캐터에 통합되어 피싱 사이트나 스캠 계정으로부터의 사용자를 보호하는 기능을 제공하기 시작합니다.




사실 이전 버전의 스캐터 데스크탑에서도 별도로 월렛이나 툴킷 사이트에 접속하지 않고도 계정에 연결된 키쌍을 변경하거나 계정 자원(RAM/CPU/NET)관리, 토큰 전송, 신규 계정 생성, 프록시 투표 설정 등 다양한 기능을 사용할 수 있는 기능을 제공했었으나 복잡한 UI로 인해 실제로 초보자들이 사용하기 어려운 단점이 있었습니다.

그러나, 이번 버전부터는 별도로 디테일한 설명이나 가이드 없이도 많은 분들이 직관적으로 사용할 수 있을 것으로 보입니다. 개인적으로도 이번 버전의 UI는 정말 섹시하다는 말 밖에 안 나오네요 :)

기존 버전에 익숙해져서 신규 업데이트 버전을 설치하지 않는 분들이 참 많은데, Meteoric은 망설이지 마시고 설치하실 것을 강추합니다! 정말 후회가 없으실 겁니다.










스캐터 Meteoric에서는 정말 간편하게 프록시 설정도 가능합니다. 그리고 이 프록시 설정과 가능하여 정말 섹시한 기능이 하나 숨겨져 있는데 그것이 바로 투표 가중치 자동감소의 방지 기능입니다.

현재 EOS 메인넷에서 BP투표에 참여하는 모든 계정들은 주기적으로 투표를 업데이트 하지 않는 경우 투표 가중치가 감소하는 Vote decay를 적용받게 됩니다. 따라서 투표에 참여하고 주기적으로 업데이트를 하지 않는 경우에는 나의 보팅파워가 자연감소하는 효과가 발생합니다.

그러나 스캐터 Meteoric을 통해 프록시 설정을 하시는 경우, 일주일마다 보팅 리프레쉬 기능을 제공하여 내 보팅 파워의 자동감소를 방지하는 역할을 자동으로 수행해줍니다.

[참고] 디온의 프록시에 간편하게 투표를 위임하는 방법




https://www.alohaeos.com/vote/proxy/proxies.info


> 스캐터 로그인 → Wallet → 계정명 클릭 → Proxy → 프록시 계정명(proxies.info) 입력 → Set Proxy 클릭 → Allow 클릭





엄청나게 빠른 개발 및 업데이트 속도로 유명했던 제임스 형제가 정말 오래간만에 11번째 버전의 스캐터를 내놓았습니다. 이번 Scatter Meteoric은 심지어 퍼블릭 테스트 버전에서도 한 단계 더 진화된 정말 섹시한 작품이라고 표현할 수 있을 것 같네요 :)

[쉐어투스팀(Share2steem)이란?]
쉐어투스팀은 트위터, 인스타그램, 미디엄, 유튜브, 트위치 등 스티밋의 계정과 연결된 다른 SNS의 글이나 영상, 사진 등의 포스팅을 스티밋 피드로 업로드할 수 있는 써드파티앱니다. 사전에 쉐어투스팀을 통해 다른 SNS채널의 계정을 연결시켜 놓은 상태라면, 언제든지 간단한 해시태그 입력만으로 다른 SNS에 업로드한 글을 그 즉시 스티밋에도 업로드하실 수 있습니다.
